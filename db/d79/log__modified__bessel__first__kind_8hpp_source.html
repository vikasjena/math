25 /* Log of the modified Bessel function of the first kind,
26  * which is better known as the Bessel I function. See
27  * modified_bessel_first_kind.hpp for the function definition.
28  * The derivatives are known to be incorrect for v = 0 because a
29  * simple constant 0 is returned.
30  *
31  * @tparam T1 type of the order (v)
32  * @tparam T2 type of argument (z)
33  * @param v Order, can be a non-integer but must be at least -1
34  * @param z Real non-negative number
35  * @throws std::domain_error if either v or z is NaN, z is
